How?

Used Technologies:
Frontend:

  1. HTML
  2. CSS
  3. JavaScript
  4. Bootstrap
  5. AJAX
  6. JSON
  7. Jinja

Backend:

  1. Python
  2. SQLite Database
  3. SQLAlchemy (ORM)(Object Relational Mapping)
  4. Flask
  5. Flask-CORS
  6. API Endpoints
  7. HTTPOnly Cookies
    8. To minimize the risk of XSS exploits
  8. Authentication
    9. Using cookies, username and password
  9. Authorization
    10. The user can only control the products that he posted
    I might use RBAC model in the future, But I didn't use it here
  10. JWT (JSON Web Tokens)
    11. In each cookie there is a jwt token, to make the authentication
  11. UnitTest
    12. To Test Python functions and SQLAlchemy Models
    (I used 112 Tests)
  12. Postman Collection
    13. To Test API Endpoints
    (I used 75 Requests)
  13. Docker

Other Technologies:

  1. Git
  2. Github
